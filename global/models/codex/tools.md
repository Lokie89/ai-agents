# 도구 목록

## 사용할 수 있는 도구

- 파일 검색: `rg`, `rg --files`
- 파일 확인: `Get-Content`, `Select-String`, `Get-ChildItem`
- 코드 변경: `apply_patch`
- 검증: 프로젝트가 제공하는 테스트, 빌드, 린트, 타입 체크 명령
- 버전 확인: `git status`, `git diff`, `git log`
- 웹 확인: 최신 정보나 공식 문서 확인이 필요한 경우

## 주의해서 사용할 도구

- 패키지 설치 명령: 필요한 경우에만 실행하고 이유를 남긴다.
- 장시간 실행되는 개발 서버: 사용자가 확인할 URL이 필요할 때 실행한다.
- 자동 포맷터: 변경 범위가 넓어질 수 있으므로 대상 파일을 확인한다.

## 금지된 도구와 행동

- 사용자 승인 없는 파괴적 명령: `git reset --hard`, 강제 삭제, 대량 이동
- 관련 없는 파일을 대상으로 한 자동 포맷팅
- 비밀값 출력 또는 저장
- 검증 우회를 위한 임시 코드
- 실패한 명령을 성공한 것처럼 기록하는 행위

## 도구 선택 기준

- 저장소에 이미 있는 명령과 도구를 우선 사용한다.
- 같은 결과를 낼 수 있다면 변경 범위가 작고 추적이 쉬운 도구를 선택한다.
- 네트워크나 권한 상승이 필요한 작업은 필요성과 영향을 명확히 한다.
